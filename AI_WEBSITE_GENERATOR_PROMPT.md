# Prompt: 3D Interactive Developer Portfolio Website

Create a **modern 3D interactive developer portfolio website** with a **creative concept using a desk table lamp**.

## Main Concept

- The homepage shows a **dark room with a 3D desk setup**.
- A **table lamp is placed on the desk**.
- When the **lamp is OFF**, the screen is dark and nothing is visible.
- When the **user clicks the lamp to turn it ON**, the **light reveals the developer portfolio** with smooth animation.
- Turning the lamp OFF hides the portfolio again.

## Design & UI

- Modern **3D UI design**
- **Dark theme with glowing neon accents**
- Beautiful **background animation (particles, gradient waves, or floating shapes)**
- Smooth **scroll animations**
- **Glassmorphism or neumorphism UI**
- Use **Three.js or React Three Fiber for 3D elements**
- Use **Framer Motion for animations**

## Special Effects

- **Lamp toggle interaction**
- **Light rays revealing content**
- **3D floating objects (code icons, laptop, phone)**
- Mouse **parallax effects**
- Smooth **page transitions**
- **Cursor glow effect**

## Technologies

Use modern technologies:

- **React / Next.js**
- **Three.js / React Three Fiber**
- **Tailwind CSS**
- **Framer Motion**

- Desktop
- Tablet

Optimize the layout and 3D animations so they work smoothly on all devices.
